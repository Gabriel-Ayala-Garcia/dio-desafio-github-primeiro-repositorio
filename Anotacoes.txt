Alguns atalhos para não esquecer:

dir : mostra as pastas e arquivos do diretório
cd (dir) : muda o diretório
ls : lista do diretório (melhor do que dir)
Ctrl + l : limpa tela
ls -a : mostra arquivos ocultos

echo > (arquivo).(extensão) : cria arquivo

==============================================

Atalhos Git

git commit -m "(mensagem)"
git config --global user.email"(email)" | --global user.name"(nome)"
	>configura o email e nome do commit
git clone : leva repositorio para um pasta
git status : verifica os arquivos commitados ou não

Arquivos criados/modificados 
	> git add (arquivo) ou * (tudo) : vai para modificado
	> git commit -m : commit tudo

git config --list : todas as modificacoes do commit

git push : modifica definitivo no online
